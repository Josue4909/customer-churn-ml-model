# Customer Churn Machine Learning Model

## 📊 Overview
This project builds a machine learning model to predict customer churn using Python and scikit-learn.

## 🛠 Tools Used
- Python
- pandas
- scikit-learn
- matplotlib

## 📈 Model Workflow
- Prepared dataset
- Trained logistic regression model
- Evaluated accuracy
- Visualized feature importance
