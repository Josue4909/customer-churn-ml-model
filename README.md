# Customer Churn Machine Learning Model
This project demonstrates a basic machine learning model built using Python and scikit-learn to predict customer churn.

## Tools Used
- Python
- pandas
- scikit-learn

## Project Steps
- Create a sample customer dataset
- Train a logistic regression model
- Predict customer churn
- Evaluate model accuracy

## Purpose
This project demonstrates basic machine learning skills including data preprocessing, model training, and evaluation.
